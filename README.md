# ZAMANAT — Цифрлық білім беру және тарихнама платформасы 🚀

**ZAMANAT** — это современная веб-платформа, разработанная для цифровизации образовательных материалов, интерактивного изучения истории Казахстана, работы с хронологией, картами и электронными школьными учебниками. 

Интерфейс выполнен в единой дизайн-системе **Premium Apple/Stripe Light Orange Concept** — адаптивный, минималистичный дизайн с использованием фирменного оранжевого акцента, эффектов стекломорфизма (`backdrop-filter`) и плавных нативных анимаций.

---

## 📱 Основные модули и структура проекта

Сайт построен по модульной архитектуре. Каждый HTML-файл отвечает за отдельный независимый функционал приложения:

1. **`index.html`** — Главная страница платформы. Включает академический прелоадер (умный индикатор загрузки на основе `document.readyState`), презентацию возможностей, интерактивную систему навигации и баннеры быстрого доступа.
2. **`books.html`** — Цифровая библиотека школьных учебников. Обладает продвинутой системой фильтрации (по классам от 5 до 11 и по предметам). Интегрирован кастомный PDF-ридер: на ноутбуках учебники открываются прямо на сайте во фрейме без перезагрузки, а на смартфонах — открываются нативно на весь экран.
3. **`chronology.html`** — Цифровая хронология «Ұлы Дала Шежіресі». Уникальный таймлайн-интерфейс со сквозным скроллом, интерактивными карточками исторических периодов и боковым стрим-трекером для быстрого перемещения по векам.
4. **`karta.html`** — Интерактивная карта сакральных мест Казахстана. Построена на базе движка **Leaflet.js** и кастомных слоев OpenStreetMap. Содержит маркеры исторических локаций с боковой панелью (Side Drawer), выдвигающейся при клике на объект.
5. **`100data.html`** — Сто важнейших исторических дат. Хронологический реестр событий с динамическим рендерингом через JavaScript и анимацией появления элементов по мере скролла на базе `IntersectionObserver`.
6. **`quiz.html`** — Интерактивные флэш-карточки для самопроверки (аналог Quizlet). Логика построена на чистом JS: переворот карточки по клику, отметки «Білемін» (Знаю) для исключения из колоды, сортировка по эпохам и интерактивный счетчик прогресса.
7. **`search.html`** — Пайдалы материалдар мен Глоссарий. Терминологический словарь с умным поиском в реальном времени (Real-time Filter по мере ввода текста).
8. **`privacy.html` & `rules.html`** — Юридические соглашения (Политика конфиденциальности и Условия использования) с интегрированным переключателем языков (KZ/RU).

## 🛠 Стек технологий

* **Frontend:** Чистый HTML5, CSS3 (Modern Flexbox / CSS Grid, CSS Variables для тем оформления).
* **Скрипты:** Чистый JavaScript (Vanilla JS — ES6+) без тяжелых зависимостей (jQuery и др.), что гарантирует максимальную скорость загрузки.
* **Картография:** Leaflet.js (Open-source библиотека для интерактивных карт).
* **Иконки и шрифты:** FontAwesome 6.5.1, шрифтовая система Google Fonts (Inter).
* **Оптимизация:** Адаптивная верстка (Mobile First / Desktop Friendly), использование `requestAnimationFrame` для анимации прелоадера, а также `IntersectionObserver` для ленивой анимации контента.

## 🔧 Особенности и исправления (Решение критических багов)

* **Фильтр классов на ноутбуках (`books.html`):** Решена проблема сжатия и наложения элементов меню («Барлығы», «5-сынып» и др.). Стили оптимизированы с помощью отключения деформации блоков (`flex-shrink: 0`) и добавления скрытого горизонтального скролла (`overflow-x: auto`), что делает панель стабильной на экранах любого разрешения (включая MacBook и ультрабуки).
* **Умный PDF-режим:** Система автоматически определяет тип устройства пользователя. На экранах `width <= 1024px` файлы открываются прямой ссылкой во вкладке браузера во избежание сбоев мобильных браузеров, а на компьютерах — разворачивается красивое модальное окно.
* **Адаптация карты:** В файле `karta.html` внедрен триггер `map.invalidateSize()` по интервальному таймеру для того, чтобы при скрытии/раскрытии боковой панели сетка карты динамически пересчитывалась без багов и серых зон.

## 🚀 Как запустить проект локально

Проект полностью автономен и не требует сборщиков (Webpack/Vite) или серверной базы данных (Node.js/PHP). 

1. Склонируйте репозиторий или скачайте архив с файлами.
2. Откройте файл `index.html` в любом современном браузере (Chrome, Safari, Edge, Firefox).
3. *Для корректной работы интерактивной карты (`karta.html`) рекомендуется запускать проект через локальный сервер (например, расширение Live Server в VS Code).*
